# Tools

Python, shell, and other tools for day to day tasks, mostly developed with LLMs.

## Python tools

| Tool | Description |
|------|-------------|
| [emailcheck.py](py/README.md#emailcheckpy) | Email domain configuration checker for deliverability validation |
| [genimg.py](py/README.md#genimgpy) | Generate images from text prompts using Google Gemini |
| [images2pdf.py](py/README.md#images2pdfpy) | Convert JPEG/JPG images to PDF with 4 images per page in a 2x2 grid |
| [mail.py](py/README.md#mailpy) | Simple mail command to send emails, replacing the Linux mail command |
| [mfields.py](py/README.md#mfieldspy) | MongoDB field analysis tool for top-level field coverage statistics |
| [speedtest.py](py/README.md#speedtestpy) | Internet speed tests using Cloudflare's speed test endpoints |
| [tgchats.py](py/README.md#tgchatspy) | Clean up empty and dead Telegram chats |
| [tgcleanup.py](py/README.md#tgcleanuppy) | Bulk unsubscribe from Telegram channels and supergroups |
| [tgpurge.py](py/README.md#tgpurgepy) | Delete all "X joined Telegram" service messages |
